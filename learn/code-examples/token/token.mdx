# TokenModule Code Examples

Donâ€™t forget to replace `<TOKEN_CONTRACT_ADDRESS>`, `<OWNER_ADDRESS>`, `<SPENDER_ADDRESS>` and `<SPENDER_ADDRESS_2>` with your own values. 

### allowance(spender)

```tsx
// Set variable for the token module contract address which can be found after creating a token module in the dashboard
const tokenAddress = "<TOKEN_CONTRACT_ADDRESS";

// Initialize the token module with the contract address
const token = sdk.getTokenModule(tokenAddress);

const allowance = (spender:string) => {  
    token.allowance(spender)
      .then((data) => console.log(data))
      .catch((error) => console.error(error));
  };
// get the allowance for specified address
allowance("<SPENDER_ADDRESS>");
```
